# MyFirstProject
## Working With Command Prompt
To change directories do: 
'''bash
cd <diretory to change in to>
'''
To see what's available:
'''bash
dir
'''
To make a new directory:
'''bash
mkdir <name of directory>
'''
## Working with Python
To launch Python do:
'''bash
python
'''
To exit Python do:
'''python
exit()
'''
To run a script:
'''bash
python <name of the script>
'''
